# Smart Temperature Monitoring System

## Project Overview

This project was developed as part of the Embedded Systems & IoT Internship Task-1.
The system monitors environmental temperature using a TMP36 temperature sensor and Arduino Uno. When the temperature exceeds 30°C, an LED turns ON to indicate a high-temperature condition.
## Components Used
* Arduino Uno R3
* TMP36 Temperature Sensor
* LED
* 220Ω Resistor
* Breadboard
* Jumper Wires
## Working Principle
The temperature sensor continuously measures ambient temperature and sends the readings to Arduino Uno. The Arduino compares the measured temperature with a threshold value of 30°C.
* Temperature ≤ 30°C → LED OFF
* Temperature > 30°C → LED ON
## Software Used
* Tinkercad Circuits
* Arduino IDE
## Project Output
The simulation successfully demonstrated temperature monitoring and alert generation using an LED indicator.
